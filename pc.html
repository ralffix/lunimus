PrankCam

PrankCam is a fun Python app that lets you prank your friends during video calls! You can toggle Torch, Flashbang, RGB effects, and flip your camera.

How to download and use:

  1. Download the PrankCam.py file from this website.
  2. Make sure you have Python installed (Python 3.10+ recommended).
  3. Open Command Prompt (Windows) or Terminal (Mac/Linux).
  4. Install the required Python libraries by running: 
    pip install opencv-python numpy pyvirtualcam
  5. Navigate to the folder where you downloaded PrankCam.py.
  6. Run the app:
    • Windows: Double-click PrankCam.py (if Python is installed) or run:
      python PrankCam.py
    • Mac/Linux: Run:
      python3 PrankCam.py
  7. Grant camera permissions if prompted.
  8. Important: Before joining a video call, start your OBS Virtual Camera. Then in the call app, select “OBS Camera” as your video source.
  9. Use the GUI buttons to toggle Torch, Flashbang, RGB mode, or flip your camera.

Tips:

  • Flashbang mode flashes the screen on your webcam feed.
  • RGB mode adds colorful overlays.
  • Flip camera flips vertically or horizontally for funny effects.
  • Quit button closes the app safely.
